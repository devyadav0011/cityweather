# 🌆 CityWeather App

A simple and responsive weather application that allows users to check real-time weather information for any city around the world 🌍.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Displays temperature, humidity, and weather conditions
* 🌤️ Dynamic weather icons
* 📱 Responsive design (works on mobile & desktop)
* ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* OpenWeatherMap API (or any weather API you used)

---
## ⚙️ Installation & Setup

1. Clone the repository

   ```bash
   git clone https://github.com/devyadav0011/cityweather.git
   ```

2. Navigate to project folder

   ```bash
   cd cityweather
   ```

---

## 🔑 API Setup

1. Get a free API key from https://openweathermap.org/api
2. Replace your API key in the JavaScript file:

   ```js
   const apiKey = "YOUR_API_KEY";
   ```

---

## 📌 Usage

* Enter a city name in the search bar
* Click the search button
* View real-time weather details instantly
